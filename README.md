# Embedded-Deep-Neural-Network-Processing
本研究在PYNQ板上對PS端進行Verilog編寫，並完成CNN、Maxpooling、Fully Connected在
Vivado中的設計，其中大數乘法採用位移乘法運算，以確保所設計的硬體模塊能夠正確且有效地被
PYNQ板上的可程式邏輯（PL）部分使用。
